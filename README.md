Example project - Use of MQTT with NodeJS + HiveMQ

# Steps to download and configure:
- git clone git@github.com:webdevlamp/nodejs-mqtt-example.git
- cd nodejs-mqtt-example
- npm install

# To Publish
- node publish.js

# To Subscribe
- node subscribe.js
